A simple social sharing plugin that adds a list of social media sharing buttons on your website.

== Description ==

As the name says, Share To Sm is a very simple to use social media sharing plugin for your WordPress website. No registration or ID's required. It's absolutely free. Just install and get started.

= Features =
* Five different types of styles: small buttons with / without share count and the usual 32 x 32 icons with square / rounded-border / circular icons.
* A very easy to configure admin panel
* Select from the available list which social icons you want to display.
* Select where you want the buttons to appear: before content and/or after content.
* Show/Hide buttons on Archive's page and/or Post and/or Pages 
* Shortcode for developers. If you want an efficient control of where the button appears you can place the shortcode '[share-to-sm]' anywhere you want to display the buttons.
* Optional tooltip and 360deg rotation animation

== Installation ==
= via ftp =
1. Upload `Share To Sm` to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Go to Settings -> Share To Sm to add social sharing buttons on your website.
= via admin dashboard =
1. Download the .zip file of the plugin.
2. Log in to your WordPress dashboard and navigate to Plugins -> Add New.
3. Press the Upload Plugin button at the top of the page.
4. Click Choose File and select the plugin file from your computer.
5. Press Install Now.
